# 📱 iPhone Tools — 仕入れ・管理ツール一式

Apple Store でのiPhone仕入れ作業を効率化し、配送遅延（島流し）を回避するための個人用ツールセットです。

## 🔗 収録ツール

| ファイル | 役割 |
|---|---|
| `index.html` | ランチャー（起点）。全ツールへのハブ＋AI相場分析 |
| `iphone-pickup.html` | ピックアップツール（仕入れフェーズ） |
| `iphone-full.html` | 購入管理ツール（管理フェーズ） |
| `iphone-pickup-manual.html` | 取扱説明書（全10章） |

## 🔒 データの扱い（重要）

- **全データは各自のブラウザ内（localStorage）にのみ保存されます。**
- サーバーには一切送信されません。
- 同じURLを複数人が開いても、**各自のデータは各自の端末内だけ**で完結し、他人からは見えません。
- Gemini APIキーも各自のブラウザに保存されます（各自のキーを使用）。

## 🚀 デプロイ（GitHub Pages）

1. このフォルダをそのままGitHubリポジトリのルートに配置
2. Settings → Pages → Branch: `main` / `/ (root)` を選択
3. 数分後、`https://<ユーザー名>.github.io/<リポジトリ名>/` で公開

`index.html` がルートなので、URLを開くとランチャーが表示されます。

## 👥 友達と共有する方法

1. 公開URLを共有するだけ
2. 各自がURLを開く → それぞれのブラウザに独立したデータが作られる
3. **共用PC・共用ブラウザプロフィールは避ける**（データが混ざる）

## 💾 バックアップ

- 各ツールの設定タブ →「💾 バックアップ / 復元」でlocalStorage全体を保存
- Chrome/Edgeなら実ファイルへ直接保存＆上書き可能
- ブラウザのデータ削除に備えて各自で定期保存
